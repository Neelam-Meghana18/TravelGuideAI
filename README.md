# 🌍 TravelGuideAI

An AI-powered travel itinerary generator built using Streamlit and Google Gemini 2.5 Flash.  
TravelGuideAI creates personalized, structured, and day-wise travel plans based on user inputs such as destination, duration, and interests.

---

## 🚀 Overview

TravelGuideAI simplifies trip planning by automatically generating customized travel itineraries using Generative AI.

Instead of manually researching tourist attractions, food options, and organizing schedules, users can simply enter their travel details and receive a complete itinerary within seconds.

The application integrates the Gemini 2.5 Flash model via the Google GenAI SDK and is deployed on Streamlit Community Cloud.

---

## ✨ Features

- 🌎 Destination-based itinerary generation  
- 📅 Day-wise structured travel plan  
- 🍽 Local food recommendations  
- 🧳 Interest-based customization  
- 📥 Downloadable itinerary  
- ☁️ Free cloud deployment  
- 🔐 Secure API key management  

---

## 🛠 Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **AI Model:** Gemini 2.5 Flash  
- **Deployment:** Streamlit Community Cloud  
- **Version Control:** Git & GitHub  

---

## 🧠 How It Works

1. User enters destination, number of days, and interests.  
2. A structured prompt is dynamically generated.  
3. The prompt is sent to Gemini 2.5 Flash via API.  
4. The AI processes the request.  
5. A personalized itinerary is generated and displayed in the web application.
